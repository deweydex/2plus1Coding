## What's Included

### For Students
- **5 Project Options**: Tic-Tac-Toe, Word Guessing, RPG Battle, Image Processing, Machine Learning
- **Two Learning Paths**: Web-based (Google Colab + Claude.ai) or Desktop (Claude Code CLI)
- **Starter Code**: Beginner and advanced versions for each project
- **Prompting Guide**: How to work effectively with AI coding assistants

### For Teachers
- **Facilitation Guide**: Classroom management for both web and desktop versions
- **Pedagogical Framework**: Grounded in constructivist learning theory
- **Project Overview Page**: Helps students choose appropriate projects
- **Timing Guidance**: 80-90 minute session structure

## Quick Start

1. **For Students**: Visit the website and choose your learning path (web or desktop)
2. **For Teachers**: Review the [For Teachers](https://deweydex.github.io/2plus1Coding/for-teachers.html) page
3. **To Contribute**: Open an issue or submit a pull request

## Learning Approach

- **Constructivist pedagogy**: Students build understanding through doing
- **Pair programming**: Collaborative learning in pairs
- **Incremental development**: Test after each small addition
- **AI as tool**: Students maintain control, AI provides assistance
