# Google Colab Notebooks

This directory contains Google Colab notebook versions of the starter code for web-based learning.

## How to Use

### Option 1: Open Directly in Colab
1. Go to [Google Colab](https://colab.research.google.com/)
2. Click File â†’ Open Notebook
3. Select the "GitHub" tab
4. Enter the repository URL: `deweydex/2plus1Coding`
5. Select the branch and navigate to `colab-notebooks/`
6. Click on the notebook you want to open

### Option 2: Use Direct Links
Once pushed to GitHub, you can create direct "Open in Colab" links:
```
https://colab.research.google.com/github/deweydex/2plus1Coding/blob/main/colab-notebooks/NOTEBOOK_NAME.ipynb
```

## Creating More Notebooks

To convert a Python starter file to a Colab notebook:
1. Create a new `.ipynb` file in this directory
2. Follow the JSON structure shown in existing notebooks
3. Break code into logical cells with markdown explanations
4. Add suggested prompts for Claude.ai
5. Include workspace cells for students to add their code

## For Teachers

These notebooks work well with Claude.ai because:
- Students can run code immediately without installation
- Easy to copy code blocks to Claude.ai for help
- Markdown cells provide inline instructions
- All libraries pre-installed in Colab environment
- Free GPU available for ML projects
